# Universal Problem Optimizer Agent 演示视频脚本

适合录制 3 到 5 分钟 MP4。

## 1. 开场介绍（约 30 秒）

大家好，本视频演示项目 Universal Problem Optimizer Agent，中文名是通用问题优化智能体。系统基于公开可访问的大模型 API，或在无 API Key 时使用 mock 模式，通过提示词工程和轻量工具调用，自动完成问题诊断、提示词优化、任务拆解、答案生成和 PDF 报告输出。

## 2. 展示公开访问链接（约 20 秒）

打开已部署的前端页面，例如 Vercel 地址。说明后端部署在 Render，前端通过环境变量 `VITE_API_BASE_URL` 调用后端 API。

## 3. 输入示例问题（约 40 秒）

在问题输入框中输入：

```text
我想写一份关于“多模态大模型在心理健康辅助诊断中的应用”的研究方案，但不知道怎么拆解研究背景、技术路线、创新点和实验设计，请帮我优化这个问题并给出完整解决方案。
```

任务类型选择“科研类”。

## 4. 点击智能体优化（约 30 秒）

点击“开始优化”。镜头停留在运行日志区域，展示系统依次执行：

- 正在分析问题；
- 正在优化提示词；
- 正在拆解任务；
- 正在调用文本摘要和关键词工具；
- 正在生成最终答案；
- 正在生成 PDF 报告。

## 5. 展示优化结果（约 70 秒）

依次讲解页面输出：

- 原始问题分析：说明问题缺少约束、输出格式和研究边界；
- 问题优化版本：展示系统生成的高质量提示词；
- 问题拆解步骤：展示多个子任务；
- 推荐求解路径与最终答案：展示研究背景、技术路线、创新点、实验设计和伦理风险。

## 6. 下载并打开 PDF（约 50 秒）

点击“下载 PDF 报告”。打开下载后的 PDF，展示报告包含：

- 项目标题；
- 用户原始问题；
- 问题诊断结果；
- 优化后的提示词；
- 任务拆解过程；
- 工具调用过程；
- 最终答案；
- 总结与改进方向。

## 7. 总结提交条件（约 30 秒）

本项目满足比赛或作业提交要求：

- 系统可公开访问；
- 使用公开大模型 API，支持 OpenAI 和通义千问；
- 无 API Key 时可用 mock 模式演示；
- 能解决一个具体问题；
- 能自动生成 PDF 求解文档；
- 整个流程适合录制 MP4 演示视频；
- 最终可提交访问链接、MP4 视频和 PDF 报告。
